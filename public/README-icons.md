# PWA Icon Generation

To create proper PNG icons for the PWA, you can:

1. Use an online converter to convert the SVG icons to PNG:
   - icon-192x192.svg → icon-192x192.png
   - icon-512x512.svg → icon-512x512.png

2. Or use a design tool like Figma, Sketch, or Adobe Illustrator to create custom icons

3. Recommended tool: https://realfavicongenerator.net/

Place the generated PNG files in the `/public` directory.
